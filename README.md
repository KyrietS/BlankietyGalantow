# Blankiety Galantów

Blankiety Galantów jest karcianą grą przeglądarkową, która umożliwia prowadzenie rozgrywek przez Internet pomiędzy użytkownikami.

## Uruchomienie
W folderze głównym repozytorium uruchom:
```
$ python -m blankiety_galantow
```
